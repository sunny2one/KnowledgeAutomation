휴대폰에서 설치 파일 만들기

1. 새 KnowledgeAutomation 저장소에 KnowledgeAutomationBundle.zip을 업로드합니다.
2. 점 3개 → Create new file을 누릅니다.
3. 파일 이름을 .github/workflows/android.yml 로 입력합니다.
4. KnowledgeAutomation_android.yml.txt 내용을 전부 복사해 붙여넣습니다.
5. Commit changes를 누릅니다.
6. Actions → Build Knowledge Automation APKs를 엽니다.
7. Run workflow를 누릅니다.
8. 성공 후 Artifacts에서 두 파일을 받습니다.
   - KnowledgeSearchBot-APK
   - KnowledgeAnswerAssistant-APK
9. 각각 압축을 풀고 app-debug.apk를 설치합니다.
